## Color Reference

| File Suffix | Color Name | Hex Code | Use This Color |
|-------------|------------|----------|----------------|
| `-black` | Black | `#000000` | Pure black |
| `-navy` | Navy | `#1e1b4b` | Dark indigo/navy |
| `-purple` | Purple | `#581c87` | Dark purple |
| `-green` | Green | `#064e3b` | Dark emerald green |
| `-burgundy` | Burgundy | `#7f1d1d` | Dark red/burgundy |
| `-cream` | Cream | `#fffbeb` | Light amber/cream |

## Specifications

- **Format:** JPG
- **Size:** 1200 x 1600 pixels (or 800 x 1000 minimum)
- **File Size:** < 500 KB
- **Aspect Ratio:** Portrait (3:4)
